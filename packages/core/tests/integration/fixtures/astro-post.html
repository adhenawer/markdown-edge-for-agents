Building Edge APIs with Astro

By Jane Astro · 2026-01-12

Astro's server islands make it easy to colocate static content with dynamic handlers. In this post we'll walk through a minimal setup.

Prerequisites

  • Node 20 or later
  • An Astro 4.x project
  • A Cloudflare account (free tier works)

The handler

Create a file at src/pages/api/hello.ts with a named GET export.

export const GET = () => new Response("hi");

Deploy with wrangler deploy and you're live.
