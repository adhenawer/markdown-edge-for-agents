Deploy Hugo to the Edge

Hank Hugo · 2026-02-03

Hugo builds absurdly fast static sites. Pairing it with Cloudflare Pages gives you a free, global CDN in minutes.

Steps

  1. Run hugo --minify to build the site.
  2. Push the public/ directory to a GitHub repository.
  3. Connect the repository on Cloudflare Pages.

That's all — your site is now served from hundreds of edge locations.

Static sites on the edge are the cheapest, fastest way to serve content at scale.
